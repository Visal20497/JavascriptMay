# 📘 JavaScript & TypeScript Concepts Practice

This repository contains a comprehensive collection of **JavaScript and TypeScript concepts**, examples, and implementations. It serves as a learning and revision guide covering core fundamentals, advanced topics, and common interview questions.

---

## 🚀 **Overview**

This project demonstrates:

* Core JavaScript fundamentals
* TypeScript type safety and interfaces
* Data structures (Array, Object, Map, Set)
* Functional programming concepts
* Asynchronous programming (Promises, Async/Await)
* Algorithms and problem-solving techniques

---

## 📚 **Topics Covered**

### 🔹 **1. Symbols**

* Unique and immutable primitive values
* Global symbol registry (`Symbol.for`, `Symbol.keyFor`)
* Using symbols as object keys

---

### 🔹 **2. Map & Set**

* Map operations: `set`, `get`, `has`, `delete`, `clear`
* Iteration using `keys()`, `values()`, `entries()`
* Set uniqueness and operations

---

### 🔹 **3. Parsing & Functions**

* `parseInt()` with different bases
* First-class functions
* IIFE (Immediately Invoked Function Expression)

---

### 🔹 **4. Objects**

* Object methods:

  * `Object.keys()`, `values()`, `entries()`
  * `Object.assign()`
  * `Object.freeze()` vs `Object.seal()`
  * `Object.getOwnPropertyNames()`
* Property checking with `hasOwnProperty`
* Object transformation using `Object.fromEntries()`

---

### 🔹 **5. Array Methods**

* `Array.from()`, `Array.of()`, `fill()`
* `filter()`, `find()`
* `split()` for strings
* Advanced iteration techniques

---

### 🔹 **6. Promises & Async/Await**

* Error handling with `try/catch`
* Promise methods:

  * `Promise.all`
  * `Promise.allSettled`
  * `Promise.race`
* Async API calls

---

### 🔹 **7. Spread & Destructuring**

* Object and array destructuring
* Spread operator usage
* Shallow vs Deep copy

---

### 🔹 **8. Closures & Scope**

* Closure examples
* Nested functions
* Lexical scoping

---

### 🔹 **9. Prototypes & Inheritance**

* Prototype chaining
* Extending array prototype
* Overriding built-in methods

---
